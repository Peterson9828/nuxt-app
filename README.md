# Nuxt 3 Punk-Web
#### Manuel Montalvo Tercero

## Folder

First clone repository:

```bash
git clone https://github.com/Peterson9828/punk-web.git

# directory
cd nuxt-app
```

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
```
